Class: HueWheel

HueWheel

new HueWheel(parent [, options])

Create a new instance of a Hue wheel, replacing the content of given parent element with the control.

Parameters:
Name Type Argument Description
parent String | HTMLElement

parent element or ID of parent element to turn into a control (content will be replaced with the control).
options Object <optional>

Options given as a JSON object

Properties
Name Type Argument Default Description

shadowBlur Number <optional>
 0

diameter (in pixels) of shadow
shadowColor String <optional>
 'black'

CSS color of shadow if active (see shadowBlur)
hue Number <optional>
 0

Initial hue angle [0, 360]

 1

lightness Number <optional>
 0.5

rgb Array <optional>

changeSaturation Boolean <optional>
 false

Enable control to change saturation
changeLightness Boolean <optional>
 false

Enable control to change lightness
colorSpace String <optional>
 'hsl'

Color space to use in control
showColorSpot Boolean <optional>
 true

Show current color in center of control
onChange Object | function <optional>

Function to call when current color changes
colorSpotWidth Number <optional>
 0.7

If showColor is used this is the ratio of the radius from center to the edge of hue wheel. Value can be [0.0, 1.0].
colorSpotBorder Number <optional>
 2

If showColor is used this is the width of the border. A value of 0 turns the border off.
colorSpotBorderColor String <optional>
 'black'

If showColor and colorBorder > 0 this will be the color of the border.
thicknessHue Number <optional>

Thickness of hue ring in pixels
thicknessLightness Number <optional>

Thickness of lightness ring in pixels
hueKnobSize Number <optional>
 0.1

A value [0.0, 1.0] determining the size of the hue knob relative to radius.
hueKnobColor String <optional>
 'white'

Color of hue knob.
lightnessKnobColor String <optional>
 'white'

Color of lightness knob (if ligthness ring is visible).
hueKnobColorSelected String <optional>
 '#777'

Color for hue knob when selected
lightnessKnobColorSelected String <optional>
 '#777'

Color of lightness knob when selected (if ligthness ring is visible).
lightnessRingClickable Boolean <optional>
 false

Makes lightness ring clickable (outside knob, inside ring).
hueKnobShadow Boolean <optional>
 false

Use shadow with hue knob if shadow is activated.
useKeys Boolean <optional>
 true

Enable usage of keys to control hsl when control is active.
hueKeyDelta Number <optional>
 1

Delta values to increment/decrement the hue value with when its hotkey is pressed.
saturationKeyDelta Number <optional>
 1

Delta values to increment/decrement the saturation value with when its hotkey is pressed.
lightnessKeyDelta Number <optional>
 1

Delta values to increment/decrement the lightness value with when its hotkey is pressed.
keyShiftFactor Number <optional>
 10

When the shift key is held down the delta values are multiplied with this factor when using the hotkeys.
hueKeyCodeUp Number <optional>
 33

Key code to use to increase hue value (default page up).
hueKeyCodeDown Number <optional>
 34

Key code to use to decrease hue value (default page down).
saturationKeyCodeUp Number <optional>
 37

Key code to use to increase hue value (default left arrow key).
saturationKeyCodeDown Number <optional>
 39

Key code to use to decrease hue value (default right arrow key).
lightnessKeyCodeUp Number <optional>
 38

Key code to use to increase lightness value (default up arrow key).
lightnessKeyCodeDown Number <optional>
 40

Key code to use to increase lightness value (default down arrow key).
tabable Boolean <optional>
 true

Allows the control to be selected and operated with tab key and hotkeys.
ctrlLock Boolean <optional>
 true

Allow holding CTRL key to lock hue while moving saturation slider
